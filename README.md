# BULDANG-BAB-KAKAO-i
💬 천안불당고등학교 카카오 i 오픈빌더 챗봇

## 소개
천안불당고등학교의 다양한 정보를 제공하는 카카오톡 챗봇입니다.

- 주요 기능
  - 🍚 오늘, 내일 급식 조회
  - 📘 학급 시간표 조회
  - 📅 이번 달 학사일정 조회
  - ⛅ 불당동 날씨 예보 확인
  

## 개발환경
- Visual Studio Code
```
npm install
node main.js
```
